
El presente proyecto trata de una pagina web que consulta la api News API,
para extraer y mostrar todas las noticias acerca de economia digital. 

Para usar la API gratuitamente se hace uso del plan Developer que ofrece la misma API, 
por lo que el proyecto solo correrá en la dirección localhost, asi que se recomienda
que para observar el funcionamiento de éste se corra en la dirección indicada. 

Nota: Aun cuando la pagina se alojo en un dominio, la api restringue cualquier consulta
de otro origen que no sea localhost ya que en el plan Developer la lista de origen desde
donde se hace la petición no permite otra que no sea la indicada.
